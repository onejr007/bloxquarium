## ğŸ“¦ Using the Asset System

### Adding Assets
1. Letakkan model/part/meshpart di folder `assets/models/`
2. Asset akan otomatis ter-detect dan di-register
3. Sistem akan otomatis pre-load asset untuk performance

### Basic Usage

```lua
local AssetManager = require(game.ReplicatedStorage.Shared.AssetManager)

-- Check if asset exists
if AssetManager.Exists("MyModel") then
    -- Get asset clone untuk digunakan
    local model = AssetManager.Clone("MyModel")
    model.Parent = workspace
    
    -- Atau spawn langsung dengan positioning
    local spawnedModel = AssetManager.Spawn("MyModel", Vector3.new(0, 10, 0))
end

-- Get list semua asset
local assetNames = AssetManager.GetAssetNames()
for _, name in pairs(assetNames) do
    print("Available asset:", name)
end
```

### Safe Operations (dengan Error Handling)

```lua
-- Safe operations dengan fallback
local fallback = AssetManager.CreateFallback("MyModel")
local model = AssetManager.SafeClone("MyModel", fallback)

-- Safe spawn dengan fallback
local spawned = AssetManager.SafeSpawn("MyModel", Vector3.new(0, 10, 0), workspace, fallback)
```

### Monitoring & Debugging

```lua
-- Check loading status
local status = AssetManager.GetLoadingStatus()
print("Assets loaded:", status.preloaded, "/", status.total)

-- Debug info
AssetManager.Debug()

-- Health check
local health = AssetManager.GetHealthStatus()
print("System health score:", health.healthScore)

-- Error monitoring
local errors = AssetManager.GetErrorStats()
for errorType, count in pairs(errors) do
    print(errorType, "errors:", count)
end
```

## ğŸ”§ API Reference

### Core Functions
- `AssetManager.Get(assetName)` - Get asset reference
- `AssetManager.Clone(assetName)` - Get asset clone
- `AssetManager.Spawn(assetName, position, parent)` - Spawn asset to workspace
- `AssetManager.Exists(assetName)` - Check if asset exists

### Safe Operations
- `AssetManager.SafeGet(assetName, fallback)` - Safe get with fallback
- `AssetManager.SafeClone(assetName, fallback)` - Safe clone with fallback
- `AssetManager.SafeSpawn(assetName, position, parent, fallback)` - Safe spawn with fallback

### Utility Functions
- `AssetManager.GetAll()` - Get all assets
- `AssetManager.GetAssetNames()` - Get list of asset names
- `AssetManager.GetLoadingStatus()` - Get preload status
- `AssetManager.IsLoaded(assetName)` - Check if asset is preloaded
- `AssetManager.Debug()` - Print debug information

### Error Handling
- `AssetManager.GetErrorLog()` - Get error log
- `AssetManager.GetErrorStats()` - Get error statistics
- `AssetManager.GetHealthStatus()` - Get system health status
- `AssetManager.CreateFallback(name, size, color)` - Create fallback asset

### Events
- `AssetManager.AssetAdded` - Fired when new asset is detected
- `AssetManager.PreloadComplete` - Fired when preloading is complete
- `AssetManager.ErrorOccurred` - Fired when error occurs

## ğŸ¯ Example Usage

```lua
-- Example: Spawning multiple assets
local AssetManager = require(game.ReplicatedStorage.Shared.AssetManager)

-- Wait for assets to be loaded
AssetManager.PreloadComplete:Connect(function(loaded, total)
    print("Preloading complete:", loaded, "/", total)
    
    -- Spawn some assets
    for i = 1, 5 do
        local position = Vector3.new(i * 10, 5, 0)
        AssetManager.Spawn("Tree", position)
        AssetManager.Spawn("Rock", position + Vector3.new(0, 0, 5))
    end
end)

-- Handle new assets being added
AssetManager.AssetAdded:Connect(function(assetName)
    print("New asset available:", assetName)
    -- Auto-spawn new assets for testing
    AssetManager.Spawn(assetName, Vector3.new(0, 10, 0))
end)
```

## ğŸ” Troubleshooting

### Asset Not Found
- Pastikan asset ada di folder `assets/models/`
- Check dengan `AssetManager.Debug()` untuk melihat asset yang ter-register
- Gunakan `AssetManager.SafeClone()` dengan fallback

### Performance Issues
- Check preload status dengan `AssetManager.GetLoadingStatus()`
- Monitor error rate dengan `AssetManager.GetHealthStatus()`
- Clear cache jika diperlukan dengan `AssetManager.ClearCache()`

### Debugging
```lua
-- Global access untuk debugging di console
```
